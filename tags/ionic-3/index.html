ionic 模拟器运行失败解决方案

   2018-07-04      移动开发      ionic 3 · ios simulator

提示错误： 命令行中编译项目成功，但是无法直接运行至模拟器，提示如下错误： No target specified for emulator. Deploying to undefined simulatorDevice type "com.apple.CoreSimulator.SimDeviceType.undefined" could not be found.[ERROR] An error occurred while running cordova run ios (exit code 1). 解决方案： 在项目根目录下运行如下命令 cd platforms/ios/cordova && npm install ios-sim@latest
