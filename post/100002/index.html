pm2 服务开机自启动

   2018-07-13      后端开发      node.js · pm2 

pm2 start index.js —name app_name
pm2 save
pm2 startup(按照CLI提示复制相应命令并执行)

参考： https://pm2.io/doc/en/runtime/guide/startup-hook/#installation
